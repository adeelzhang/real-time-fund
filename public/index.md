# 估基

估基是一款面向网页端和移动端的基金估值与持仓管理工具。它将基金搜索、实时估值参考、日内走势、历史净值、前十大持仓、收益记录和全球行情整理在同一个界面中。

估基不提供基金交易、开户、代销或投资顾问服务。页面中的估值、行情和收益计算仅用于信息整理与个人参考，不构成投资建议、收益承诺或交易信号。

## 主要能力

- 按基金名称或代码搜索并管理自选基金。
- 查看交易时段内的实时估值参考和日内走势。
- 查看历史净值、前十大持仓及相关证券走势。
- 记录份额、成本、交易和分红方式，计算持仓金额与收益。
- 使用分组、表格和卡片视图管理不同清单。
- 登录后在不同设备之间同步持仓与显示设置。
- 查看主要指数、市场板块和全球市场行情。

## 数据与计算边界

实时估值是根据公开持仓与市场行情形成的参考结果，不是基金管理人确认的单位净值。前十大持仓通常来自基金定期报告，可能滞后于基金当前实际持仓。行情、净值和持仓数据可能受上游接口、交易时区、节假日及网络状态影响而延迟或缺失。

持仓金额和收益结果依赖用户录入的份额、成本、交易及分红信息。基金净值、资产和收益应以基金管理人、托管人、交易所或销售机构发布的正式信息为准。

## 数据与隐私

未登录时，基金清单、持仓和偏好默认保存在当前浏览器。登录后，用户可通过估基配置的云端服务进行跨设备同步。账户、个人持仓和内部 API 不是公开内容，也不应被搜索引擎或 AI 服务索引。

估基的公开访问统计仅发送到估基自有服务器，不接入第三方统计、错误上报或外部社群服务。
